# Skin

Returns skin in .png format.

[GET] `https://assets.zaralx.ru/api/v1/minecraft/vanilla/player/skin/<player_uuid>`

## Examples
[GET] `https://assets.zaralx.ru/api/v1/minecraft/vanilla/player/skin/11fd94e9706b486e827460c55a55c01b`
