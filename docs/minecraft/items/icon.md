# Inventory Icon

Returns rendered minecraft icon with 256x256 resolution.

[GET] `https://assets.zaralx.ru/api/v1/minecraft/vanilla/item/<item_id>/icon`

## Examples
[GET] `https://assets.zaralx.ru/api/v1/minecraft/vanilla/item/carrot/icon`
